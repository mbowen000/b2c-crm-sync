B2CHttpTestCalloutMockGenerator

public class B2CHttpTestCalloutMockGenerator implements HttpCalloutMock
This is a helper class for test cases that generates mocks for web-service based unit tests. It generates the mock success and failure http responses for B2C API Interactions.

Abraham David Lloyd
May 10th, 2020
