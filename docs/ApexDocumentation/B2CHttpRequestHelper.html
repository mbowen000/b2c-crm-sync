B2CHttpRequestHelper

public with sharing class B2CHttpRequestHelper
This class is a helper class used to create stock HttpRequest definitions that are used by REST interaction classes to interact with B2C Commerce REST APIs.

Abraham David Lloyd
May 8rd, 2020

Methods

  • getAccountManagerAuthRequest
    Generates the httpRequest to authenticate an against the B2C Commerce Account Manager and retrieve an accessToken for REST API access.
  • getActiveCustomerPromotionsRequest
    Generates the httpRequest to retrieve the B2C Commerce Customer Profile through the Shop API meaning that the retrieval is done in the context of an authenticated customer
  • getAllSitesRequest
    Generates the httpRequest to retrieve all storefront sites.
  • getBusinessManagerAuthRequest
