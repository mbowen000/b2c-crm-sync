B2CBaseAttributeAssignment

public abstract with sharing class B2CBaseAttributeAssignment
This is the base class used to perform attribute assignments between source B2C Commerce API responses and target Salesforce SObjects (ex. Contact).

Abraham David Lloyd
May 24th, 2020

Methods

  • applyMappedFieldValues
    This method takes a typed sObject, Map / JSONParse representations of the object definition to process, and a collection of fieldMappings identifying which fields to update. It then evaluates the objects and fieldMappings to determine which fields to update -- and applies those updates to the typed sObject.
  • assignB2CAttributeValue
    Helper function that assigns b2c attributes and values back to a parent object. This function evaluates the field mapping attribute and casts the attribute value to the appropriate data format for assignment to the SObject.
  • assignMissingFieldValues
    Helper function to assign missing fieldValues present on one sObject to a target sObject of the same type.
  • doesFieldExist
    Helper function to determine if a given field exists within an object schema.
  • doesFieldExistInJSONObject
    Helper function to determine if a given field exists within a parsed JSON Object
  • getB2CAttributeBooleanValue
    Helper function that converts a boolean property from its JSON source to the typed value that can then safely be assigned to a source object.
  • getB2CAttributeDateTimeValue
    Helper function that converts a date-time property from its JSON source to the typed value that can then safely be assigned to a source object.
  • getB2CAttributeDateValue
    Helper function that converts a date property from its JSON source to the typed value that can then safely be assigned to a source object.
  • getB2CAttributeDecimalValue
    Helper function that converts a decimal property from its JSON source to the typed value that can then safely be assigned to a source object.
  • getB2CAttributeIntegerValue
    Helper function that converts an integer property from its JSON source to the typed value that can then safely be assigned to a source object.
  • getB2CAttributeStringValue
    Helper function that converts a string property from its JSON source to the typed value that can then safely be assigned to a source object.
  • getFieldDisplayType
    Helper function to return the fieldDisplayType for a given object schema
  • getSchemaMap
    Helper function to return the schemaMap for a given sObject
  • getUpdatedFieldMappings
    This method compares the "before" and "after" version of a processed sObject and evaluates if any updates were made to the record. It does this by iterating over the collection of field mappings and comparing the field values in the "before" and "after" objects. It collects each of the fieldMappings representing an update / difference between the "before" and "after" sObject versions.
  • hasSObjectBeenUpdated
    This method compares the "before" and "after" version of a processed sObject and evaluates if any updates were made to the record. It does this by iterating over the collection of field mappings and comparing the field values in the "before" and "after" objects. If one field is found to have a different value -- then the updated flag is set to true.
  • removePersonAccountProperties
    Helper function to remove the account-specific PersonAccount properties from a Contact object; we remove these values as they will prevent the Contact from being updated directly if the Person contactModel is enabled
  • updateChangedFieldValues
    Helper function to update changed fields found between the sourceObject and its updated / synchronized targetObject.

getUpdatedFieldMappings

public static List<B2C_Integration_Field_Mappings__mdt> getUpdatedFieldMappings(SObject originalSObject, SObject processedSObject, List<B2C_Integration_Field_Mappings__mdt> fieldMappings)
This method compares the "before" and "after" version of a processed sObject and evaluates if any updates were made to the record. It does this by iterating over the collection of field mappings and comparing the field values in the "before" and "after" objects. It collects each of the fieldMappings representing an update / difference between the "before" and "after" sObject versions.
Parameters
originalSObject
{SObject} Represents a clone of the original sObject being processed (before updates)
processedSObject
{SObject} Represents the processed sObject (with updates; after)
fieldMappings
{List<B2C_Integration_Field_Mappings__mdt>} Represents the collection of field mappings to leverage
Return Value
{List<B2C_Integration_Field_Mappings__mdt>} Returns the collection of field mappings that represent updated sObject properties
